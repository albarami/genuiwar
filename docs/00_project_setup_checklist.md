# GenUIWar — Project Setup Checklist

## Purpose
Track what must exist before serious implementation begins.

## Must exist
- docs backbone
- Cursor rules
- AGENTS.md
- README.md
- .env.example
- repo structure
- environment strategy
- stack decisions
- synthetic-first development agreement

## Confirm before coding
- repo connected to GitHub
- local environment works
- Python and Node available
- backend stack chosen
- frontend stack chosen
- package managers chosen
- no-free-facts rule accepted
- clarification behavior accepted
- dynamic schema handling accepted
- agent roles defined
- UX target documented
- first Cursor brief ready

## Definition of setup complete
Setup is complete only when:
1. repository works
2. docs exist
3. Cursor rules exist
4. environment contract exists
5. first build brief exists
